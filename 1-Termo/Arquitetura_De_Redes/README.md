# 🌐 Arquitetura de Redes

### 📑 Descrição
Estudo focado na infraestrutura de comunicação de dados, cobrindo desde os componentes físicos até os protocolos que permitem o funcionamento da internet e redes locais.

### 🧠 Principais Tópicos
- **Modelo OSI & TCP/IP:** Camadas e fluxos de dados.
- **Protocolos:** HTTP, HTTPS, DNS, DHCP, FTP, SSH.
- **Topologias:** Estrela, Malha, Barramento e configurações de LAN/WAN.
- **Segurança:** Noções de Firewall e endereçamento IP (IPv4/IPv6).

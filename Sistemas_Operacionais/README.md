# 🖥️ Sistemas Operacionais

### 📑 Descrição
Exploração do funcionamento do software que gerencia o hardware, com foco em administração e comandos.

### 🧠 Principais Tópicos
- **Gerenciamento de Processos:** Como o SO lida com múltiplas tarefas.
- **Gestão de Memória:** Paginação e memória virtual.
- **Sistemas de Arquivos:** NTFS, FAT32, EXT4.
- **Terminal e CLI:** Comandos avançados em Linux (Bash) e Windows (CMD/PowerShell).

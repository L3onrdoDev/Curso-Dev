Este repositório é dedicado ao armazenamento de exercícios, desafios e projetos desenvolvidos durante as aulas de Python no curso de **Desenvolvimento de Sistemas (SENAI)**.

---

### 📂 Estrutura do Repositório

Para facilitar a navegação, os arquivos estão organizados por temas ou módulos:

* **`01-Logica/`**: Exercícios básicos de variáveis, tipos de dados e operadores.
* **`02-Estruturas-Condicionais/`**: Práticas com `if`, `elif` e `else`.
* **`03-Lacos-Repeticao/`**: Exemplos de `for` e `while`.
* **`04-Listas-Dicionarios/`**: Manipulação de coleções de dados.
* **`Projetos/`**: Mini-projetos integradores (ex: Calculadora, Jogo do Termo, etc).

# 📝 Levantamento de Requisitos

### 📑 Descrição
Focado na fase inicial do desenvolvimento de software: entender a dor do cliente e transformar em documentação técnica.

### 🧠 Principais Tópicos
- **Elicitação de Requisitos:** Técnicas de entrevista e observação.
- **Requisitos Funcionais vs. Não-Funcionais:** O que o sistema faz vs. como ele deve ser.
- **Documentação:** Criação de Casos de Uso e User Stories.
- **Metodologias Ágeis:** Introdução ao SCRUM e Kanban.
